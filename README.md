# STM32 LED Blinking Project

This project demonstrates basic LED blinking using STM32 microcontroller
and STM32CubeIDE.

## Hardware Used
- STM32 Development Board
- On-board LED
- USB cable

## Software Used
- STM32CubeIDE

## Working
The LED is toggled using GPIO and delay function inside an infinite loop.
